How AI Mining Works

1

Challenge

A deterministic AI prompt is derived from the previous block hash and current height. Every block gets a unique challenge.

2

Inference

The miner's local Ollama instance processes the challenge through a language model, generating a unique AI response.

3

Proof

The response is hashed and embedded in the coinbase transaction as an OP_RETURN output — a permanent AI proof in the block.

4

Mine

Standard Scrypt proof-of-work continues. The AI proof becomes part of the block's identity through the Merkle root.

# Quick start
tar xzf shardcoin-core-linux-aarch64.tar.gz
cd shardcoin-core

# Start the node
./shardcoind -daemon

# Check status
./shardcoin-cli getblockchaininfo
./shardcoin-cli getaiinfo

# Mine with AI (requires Ollama running)
ollama serve &
./shardcoin-cli createwallet "main"
./shardcoin-cli -generate 1
