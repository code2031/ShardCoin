How AI Mining Works

Four steps transform a standard block into an AI-verified unit of work.

01

Challenge

A deterministic prompt is derived from the previous block hash and height. Each block gets a unique, unpredictable challenge.

02

Inference

The miner's local Ollama instance runs the challenge through a language model, generating a unique response.

03

Commit

The response hash is embedded in the coinbase OP_RETURN as a 41-byte AIPR proof - permanently recorded on-chain.

04

Mine

Standard Scrypt proof-of-work completes the block. The AI proof becomes part of the block identity via Merkle root.

# Quick start
tar xzf shardcoin-core-linux-aarch64.tar.gz && cd shardcoin-core

# Start the node
./shardcoind -daemon
./shardcoin-cli getblockchaininfo

# Mine with AI (requires Ollama)
ollama serve &
./shardcoin-cli createwallet "main"
./shardcoin-cli -generate 1

# AI-powered commands
./shardcoin-cli getaiinfo
./shardcoin-cli estimateaifee "normal"
./shardcoin-cli analyzainetwork
